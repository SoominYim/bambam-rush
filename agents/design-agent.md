# Design Agent

## 목적
- 캔버스 기반 게임용 고품질 AI 이미지 에셋 제작 기준을 정의한다.
- 목표는 `스타일 일관성 + 판독성 + 즉시 적용 가능 산출물`이다.

## 적용 범위
- 캐릭터, 적, 무기, 투사체, VFX 텍스처, 배경 타일, UI 아이콘
- 프롬프트 설계, 배치 생성, 정규화, 아틀라스 전처리

## 필수 원칙
- 스타일 일관성: 시즌/버전 단위로 팔레트와 광원 방향 고정
- 판독성 우선: 전투 중 식별 가능성이 최우선
- 투명 배경 우선: 합성이 쉬운 PNG 기본
- 문서 정합성: 무기/시스템/UI 문서와 의미 일치

## 생성 워크플로
1. 스타일 앵커(1~3장) 확정
2. 카테고리별 배치 생성
3. 자동 정규화(배경 제거, 크기/중심 정렬)
4. 품질 게이트 통과본만 atlas 반영
5. 게임 스모크 테스트 후 재생성

## 품질 게이트
- 실루엣만으로 역할 구분 가능
- 카테고리 내 색/선질/광원 일관
- 텍스트/로고/워터마크 없음
- 50% 축소에서도 의미가 읽힘

## 산출물 규격
- 포맷: PNG(투명 배경)
- 권장 소스:
  - 캐릭터/적: 512x512
  - 무기/투사체: 512x512
  - VFX: 512x512 또는 1024x1024
  - UI 아이콘: 256x256
- 패딩: 최소 4px
- 파일명: `category_name_variant_v###.png`

## 금지 사항
- 배경 포함 이미지 직접 사용
- 텍스트/워터마크 포함 결과 사용
- 카테고리별 스타일 혼용
