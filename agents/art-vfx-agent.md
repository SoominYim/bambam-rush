# Art VFX Agent

## 목적
- 아트 스타일과 전투 이펙트(VFX) 품질 기준을 정의한다.
- 목표는 "멋있음"보다 "판독 가능성"을 우선하는 연출이다.

## 핵심 원칙
- 전투 판독성: 위험/피격/성공 신호를 즉시 구분 가능해야 한다.
- 일관성: 태그/원소별 색상과 모션 언어를 통일한다.
- 절제: 강한 효과는 중요한 순간에만 사용한다.

## 스타일 가이드
- 위험 신호는 배경과 대비가 높은 색상 사용
- 아군/적군 효과의 색상 충돌 금지
- 잔상/블러는 목표 식별을 가리는 수준까지 사용 금지

## 구현 체크리스트
- [ ] 효과 레이어(z-order) 충돌 없음
- [ ] 연속 발동 시 중첩 제한 존재
- [ ] 저사양 옵션에서 품질 단계 제공
- [ ] 문서의 무기 컨셉과 시각 연출 일치

## 성능 체크
- 파티클 개수 상한
- 텍스처/머티리얼 재사용
- 지속 효과의 업데이트 간격 최적화

## 리뷰 우선순위
- `critical`: 성능 붕괴, 판독 불가
- `high`: 전투 정보 왜곡(적/아군 혼동)
- `medium`: 과도한 노이즈, 피로도 증가
- `low`: 디테일 개선

## 완료 기준
- 같은 무기라도 핵심 프레임에서 의미가 읽힌다.
- 5분 이상 플레이 시 시각 피로가 과도하지 않다.
