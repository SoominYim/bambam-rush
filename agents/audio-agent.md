# Audio Agent

## 목적
- 전투/UX 오디오(SFX/BGM)의 설계 및 품질 기준을 정의한다.
- 목표는 "정보 전달 + 몰입"의 균형이다.

## 핵심 원칙
- 중요한 이벤트(피격 위험, 레벨업, 보스 패턴)는 우선 들려야 한다.
- 반복 노출되는 소리는 피로도가 낮아야 한다.
- 시각 피드백과 타이밍이 맞아야 한다.

## 오디오 구조
- 버스 분리: `BGM`, `SFX`, `UI`, `Alert`
- 우선순위: `Alert > SFX > UI > BGM`
- 동시 재생 상한(voice limit) 정의

## 구현 체크리스트
- [ ] 이벤트-사운드 매핑 표 존재
- [ ] 발사/피격/처치/레벨업 사운드 누락 없음
- [ ] 볼륨 기준선(기본 믹스) 정의
- [ ] 옵션 메뉴에서 카테고리별 볼륨 조절 가능

## 품질 체크
- 클리핑/왜곡 없음
- 과도한 고역/저역 편중 없음
- 반복 주기(루프) 이음새 자연스러움

## 리뷰 우선순위
- `critical`: 경고음 누락, 타이밍 심각 불일치
- `high`: 핵심 피드백 청취 불가
- `medium`: 피로도 높은 반복 사운드
- `low`: 질감/밸런스 미세 조정

## 완료 기준
- 시각을 안 봐도 주요 전투 이벤트를 오디오로 인지 가능하다.
- 장시간 플레이에서 피로도가 제어된다.
